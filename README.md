## 📌 Overview

StreetBase is an AI-powered real estate platform designed to help users make smarter property decisions through Machine Learning and Data Analytics.

The platform predicts house prices, compares properties, calculates EMIs, provides AI-generated interior visualizations, and delivers real-time real-estate insights through a user-friendly interface.

StreetBase also includes an intelligent virtual assistant that guides users through all platform features.

---

## 🎯 Problem Statement

Property buyers often face challenges such as:

- Lack of accurate property pricing
- Difficulty comparing multiple properties
- Complex loan calculations
- Limited access to market insights
- No centralized platform for property analysis

StreetBase solves these problems by integrating AI-powered tools into a single platform.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/StreetBase.git
cd StreetBase
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Linux / Mac

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```
